📘 第1週 — 因果・集団・ランダム

統計因果ランダムサンプリング

1. データ分析って何をしているの？

お菓子を食べたらお腹が痛くなった。原因はお菓子？それとも走りすぎた？データ分析はこの「どれが原因？」を解く探偵の仕事です。でも現実はノイズだらけ。だから数学で雑音をそぎ落とすのです。
正式な定義：データ分析とは、目的とする評価指標に対して影響を与える要因の効果を、実験誤差・観察誤差を取り除いたうえで定量的に測ることです。統計的手法なしでは、ノイズが真の効果を覆い隠してしまいます。

2. 因果 vs 統計

因果直接の原因（因）＋条件（縁）→結果（果）。『Xが起きたからYが起きた』。
統計集団全体を見て、その傾向を数値でまとめること。

3. 個人バラバラなのに、なぜ集団を調べる？

1人の子がほうれん草を食べて背が伸びた → たまたまかも。でも1,000人が食べて多くが伸びた → ホンモノの効果っぽい！集団にすると『運』が平均化されるのです。
ポイント：科学的な因果の裏付けには集団レベルの証拠が必須です。1人の患者を同時に『治療あり』『治療なし』両方にはできません。しかしランダムに分けた2集団なら、その不可能な比較の代わりになります。

4. 祈祷師の話 🔮

病気の人が祈祷師を訪ねた。祈祷師が祈った。1週間後 — 治った！祈りが効いた？

治療結果効いた？
あり改善✅ たぶん
あり変化なし❌
なし改善❌ （治療不要）
なし変化なし✅ 効いていた
⚠️ 問題：同じ人の『治療あり』『治療なし』を同時に観察できません。これが因果推論の根本的な難しさです。

5. 母集団と標本

母集団調べたい全員・全部。例：『日本の糖尿病患者全員』。
標本実際に測定する小さな一部。ここから母集団を推測します。
全数調査全員を調べる。例：国勢調査。
標本調査一部を調べて全体を推測。安くて速い。

6. ランダムサンプリング

1,000個のカラフルなあめ玉が入った袋。目を閉じて50個つかむ → ランダムサンプリング！手の中のあめ玉は袋全体とだいたい同じ色の比率になります。逆に上の赤いのだけ取ったら… 偏り（バイアス）！
なぜランダム？他のルール（例『朝来た最初の10人』）は必ず隠れた要因（年齢・職業・時間の都合）と関係します。ランダムだけが『どの要因とも関係しない』ことを保証できるルールです。ラボタブの『あめ玉シミュレーター』で体験してみよう！

7. ランダム化比較研究（RCT）

  1. 対象者を集める。
  2. ランダムにグループ分け（治療群 vs 対照群）。
  3. 各グループに異なる介入を実施。
  4. 結果を測定。
  5. 比較 → 本当に差がある？
🏆 RCTは因果証拠のゴールドスタンダード。現代医療はこの土台の上にあります。
☕ コラム：フィッシャーの3原則
  • 反復 — ばらつきを測るため繰り返す
  • 無作為化 — 条件をランダムに割り当てる
  • 局所管理 — 既知の攪乱因子をブロック化で抑える

📗 第2週 — データを概観する

度数分布表ヒストグラム四分位中央値

1. まず『概観』する理由

パズルを解く前に、ピースを机に広げて眺めますよね。第2週はその『眺める』工程。数字の山を表や絵にして、何が起きているかを目で見えるようにします。

2. データの種類（尺度）

📛 名義尺度ただの区別。血液型、好きな色。順序なし。
🥇 順序尺度順序あり、でも間隔は不明。映画の★評価。
🌡️ 間隔尺度間隔が等しいが、原点(0)に意味なし。摂氏温度。
📏 比尺度間隔等しい＋原点あり。身長、体重、個数。
前2つ（名義・順序）＝質的変数。後2つ（間隔・比）＝量的変数。尺度が意味ある統計量を決めます。例：『平均血液型』はナンセンス。

3. 度数分布表

各『階級（バケツ）』にデータがいくつ入るかを数える表。

階級度数相対度数累積度数
1.5 ≤ x < 1.630.103
1.6 ≤ x < 1.7120.4015
1.7 ≤ x < 1.8100.3325
1.8 ≤ x < 1.950.1730
⚠️ 『以上・未満』— 1.6は含む、1.7は含まない。試験で超重要！

4. 相対度数、累積、パーセンタイル

相対度数= 度数 ÷ 全データ数. 合計＝1.00
累積度数小さい階級から順に足していく。
累積相対度数相対度数の累積。『この値以下は全体の何％』が分かる。
パーセンタイルExcel: =PERCENTILE.INC(range, 0.75)

5. 四分位と中央値

データを小さい順に並べ、4等分します。

  • Q1 = 25% = 第1四分位点
  • Q2 = 50% = 中央値 (= MEDIAN)
  • Q3 = 75% = 第3四分位点
Excel: =QUARTILE.INC(range, 1), =QUARTILE.INC(range, 2), =QUARTILE.INC(range, 3)

6. ヒストグラムとスタージェスの公式

ヒストグラムは度数分布表を棒グラフにしたもの。階級数の選び方がめちゃくちゃ大事！

📐 スタージェスの公式： k = 1 + log₂(n). n=100なら k ≈ 1 + 6.64 ≈ 8階級。

7. Excel での作り方

# 方法1：分析ツール → ヒストグラム File → Options → Add-ins → Analysis ToolPak → OK Data tab → Data Analysis → Histogram Input range: your data Bin range: your class upper bounds # 方法2：FREQUENCY関数（配列数式） 1. Select output cells D2:D5 2. Type =FREQUENCY(data_range, bin_range) 3. Press F2, then Ctrl+Shift+Enter
⚠️ 分析ツールは『以下』で区切ります（『未満』ではない）。『未満』にしたければ 2 ではなく 1.9999 と入力！
